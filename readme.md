使用方法
----

- vscode

  如果打开的是文件夹，将pyi文件扔到插件同文件夹 如果只打开了文件，将pyi文件扔到编辑器打开即可


- pycharm

  将pyi文件扔在py文件同目录即可

tip:插件编写完成后无需将pyi文件扔到服务端
